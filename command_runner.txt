### Migration to terminal execution ###
- docker compose exec app php artisan migrate --force

### Run seeder with specific ###
- docker compose exec app php artisan db:seed --class=EmailTemplateSeeder
- docker compose exec app php artisan db:seed --class=NavItemSeeder


### Down docker and Build ###
- docker-compose down -v                             
- docker-compose up -d --build

### update composer ###
- docker compose exec app composer install --no-interaction

### run all seeder ###
run this inside the docker for compatibility
- php artisan db:seed

### Install Stripe PHP SDK (run once) ###
- docker compose exec app composer require stripe/stripe-php

### Clear config cache after adding Stripe keys to .env ###
- docker compose exec app php artisan config:clear
